# Daily Lessons

This directory contains all 50 days of the deep learning course.

## Structure

Each day folder follows this structure:

```
day_XX/
├── README.md              # Day overview and objectives
├── notebooks/             # Jupyter notebooks
│   └── day_XX_exercise.ipynb
├── code/                  # Python scripts (optional)
├── data/                  # Day-specific datasets (optional)
└── outputs/               # Generated plots/results
```
