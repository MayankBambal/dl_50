# Scripts Directory

Helper scripts for setup and data management.

## Available Scripts

### setup_environment.sh / setup_environment.bat
Sets up the Python environment and installs dependencies.

### download_data.sh / download_data.bat
Downloads required datasets for the course.

## Usage

**macOS/Linux:**
```bash
chmod +x scripts/setup_environment.sh
./scripts/setup_environment.sh
```

**Windows:**
```cmd
scripts\setup_environment.bat
```

## Adding New Scripts

When adding new scripts:
- Make them executable (Linux/Mac)
- Add clear documentation at the top
- Include error handling
- Test on multiple platforms if possible
